# Banking System API

RESTful Banking System built with Spring Boot.

## Features

- JWT Authentication
- Role Based Authorization
- User Management
- Account Management
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- Validation
- Global Exception Handling
- Swagger Documentation

## Technologies

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- JWT
- Swagger OpenAPI
- Maven

## API Documentation
